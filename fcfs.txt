24.	Write an R Program to write a own generic function in S4Class.
#The function setClass() returns a generator function.
student <- setClass("student", slots=list(name="character", age="numeric", GPA="numeric"))
#We can write our own method using setMethod() helper function.
#You can list all the S4 generic functions and methods available, using the function showMethod
setMethod("show","student",
function(object) {
cat(object@name, "\n")
cat(object@age, "years old\n")
cat("GPA:", object@GPA, "\n")
}
)
s <- new("student",name="John", age=21, GPA=3.5)
 s   
Output:
	John 
	21 years old
	GPA: 3.5
